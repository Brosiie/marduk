      # Sundered Coast

      **Region ID:** `sundered_coast`
      **Recommended level range:** 55–70
      **Boss:** `glade_terror`
      **Palette:** Bruise-violet · drowned-grey · spawn-pink

      ## Atmosphere

      Half-sunken Tiamat ribs jut out of the surf, each rib three storeys tall. Spawn hatches in the tidepools. The deep priests survived and resent you for it.

      ## Mobs that spawn here

        - `blood_hunter`
- `binding_construct`
- `shrine_acolyte`
