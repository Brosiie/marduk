      # Black Citadel

      **Region ID:** `black_citadel`
      **Recommended level range:** 70–85
      **Boss:** `siege_master`
      **Palette:** Coal-black · iron-rust · ember-spark

      ## Atmosphere

      The fortress where Spawn-Eaters became Citadel Priests. Bound Spawn haven't eaten in 400 years. Tier-Wardens patrol with weaponized paperwork. Ash falls indoors.

      ## Mobs that spawn here

- `blood_hunter`
- `shrine_acolyte`
