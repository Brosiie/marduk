      # Ember Steppes

      **Region ID:** `ember_steppes`
      **Recommended level range:** 28–42
      **Boss:** `corrupt_master`
      **Palette:** Ember-orange · basalt-black · sulfur-yellow

      ## Atmosphere

      Flame-Breathing apostates' last ground. Steppe Salamanders the size of horses live in the fissures. Pillar of Nergal smokes on the horizon. Dry wind, hot light.
