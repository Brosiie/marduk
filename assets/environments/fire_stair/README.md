      # Fire Stair

      **Region ID:** `fire_stair`
      **Recommended level range:** 85–100
      **Boss:** `self_that_said_yes`
      **Palette:** Lucifer-red · brass · damnation-black

      ## Atmosphere

      The basalt staircase down to Lucifer's throne. Each step is warm enough to char unbalanced footing. Ember Drakes wheel overhead like loyal dogs. Silver Negotiators offer the same deal in worse polish.

      ## Mobs that spawn here

        - `blood_hunter`
- `shrine_acolyte`
